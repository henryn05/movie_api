Users

Request URL Description Method Query Parameters Request Body Response

GET /users/{user_id} Details of a specific user GET None None JSON object representing a user
POST /users Create a new user POST None JSON object with user data (e.g., username, email) JSON object representing the newly created user
PUT /users/{user_id} Update a user's information PUT None JSON object with updated user data JSON object representing the updated user
DELETE /users/{user_id} Delete a user DELETE None None Success message or status code
